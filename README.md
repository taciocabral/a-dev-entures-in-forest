## Install the dependencies
```
make install
```

## Run the Game
```
make run
```

## Commands:
**Use the arrow keys to guide the character.**

**Attack:** `SPACE`

**Spell:** `Ctrl`

**Change weapon:** `w`

**Swap spell:** `q`

**Upgrade:** `m`

**Leave the game:**
- `Ctrl + c` in the terminal;
- **MacOS:** `cmd + w`
- **Windows:** `alt + f4`
